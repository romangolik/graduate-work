Базові положення

Проект JS-Soft генерування коду для UV-PCB-tech функціонально не є самодостанім конвертором, оскільки отримання CNC-файлу лише за описом геометрії обєктів PCB-файлів не можливе, оскільки проведення необхідних геометричних трансформації базується на встановлення групи взаємоповязаних технологічних параметрів, які задає користувач. Більш завершеною формою програмного забезпечення генерування коду для UV-PCB-tech обробки, як утиліти САМ-постпроцесора, є інтегруванням його у виробничу систему з такими варіантами використання:

  • Персональний верстат - користувач і є оператором;
  • UV-PCB-tech сервіс.

Установка

Однією з переваг розробки програми на JS є підтримка його всіма основними браузерами. FlatTCAM написано на Python, і для його роботи потрібно додатково запускати інтерпритатор, що звужує область та вносить незручності у використанні. Проект "JS-Soft генерування коду для UV-PCB- tech" як браузерний додаток дозволяє досягти поліпрофільності та високу гнучкість використання. Наприклад, він може бути використаний:

  • як автономний TCAM конвертор, достатньо лише файли html, css, js та json розмістити у локальному сховищі та запустити браузер;
  • як серверний додаток, достатньо лише залити на веб-пакет файлів на сервер та запускати дистанційно;
  • як вбудоване програмне забезпечення, що реалізується шляхом заливки веб-пакету файлів на Flash EEPROM WIFI-модуля (зокрема ESP8266 чи ESP32 SPIFFS утилітою завантаження Flash DownLoad Tool, при прошивці або OTA).

Опис використання

У варіанті використання "Персональний верстат" передбачаються такі закладки: "Головна", "Параметри", "Контроль", "CNC-управління", "Конфігуратор", "Довідка".

Головна закладка

"Головна" – панель відкриття вхідних файлів та верифікатор завантажень.

Редагування технологічних параметрів

"Параметри" – панель редагування технологічних параметрів обробки.

Верифікація CNC-файлу

Ця панель надає користувачу доступ до віджетів створення CNC-файлів – віртуальна кнопка «Generate» та верифікації отриманих результатів.

CNC-управління

"CNC-управління" – панель прямого управління верстатом.

Конфігуратор

"Конфігуратор" – панель візуалізації та редагування основних параметрів системи json-файла.
